# Personal Data Copy Page

Легкое статическое веб-приложение для быстрого копирования личных полей в один клик.

## Что делает приложение

- Показывает набор полей с синтетическими значениями по умолчанию.
- Поддерживает два режима:
  - **Редактирование**: изменение данных и настройка набора полей.
  - **Боевой режим**: одно нажатие по значению копирует его в буфер обмена.
- Поддерживает пользовательские поля.
- Сохраняет изменения только в `localStorage` браузера.

## Модель приватности

- Нет backend.
- Нет внешних API-запросов.
- Нет аналитики.
- Нет телеметрии.
- Нет внешних шрифтов и CDN.
- В репозитории только синтетические значения по умолчанию.
- Реальные данные вводятся вручную только в браузере пользователя и сохраняются в `localStorage`.

Ключ `localStorage`: `personal-data-copy-page:v1`

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация на GitHub Pages

Проект является статическим и может быть опубликован как обычный build Vite.

Важно: реальные персональные данные никогда не должны коммититься в репозиторий.

## Сброс данных из UI

В режиме **Редактирование** откройте блок **Настройки полей** и нажмите:

`Сбросить к синтетическим данным`

После подтверждения значения в `localStorage` будут заменены синтетическими значениями по умолчанию.
